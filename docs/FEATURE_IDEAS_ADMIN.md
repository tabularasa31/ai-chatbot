# Admin / Operator Interface Ideas

### [FI-012] Admin dashboard (operator view across clients)

**Идея:** Отдельный интерфейс для администратора платформы (т.е. тебя), который показывает картину по всем клиентам, а не только по одному.

**Что может включать:**
- Список всех клиентов:
  - имя, домен, статус, наличие OpenAI ключа
  - количество документов, чат-сессий, токенов за период
- Глобальный лог чатов (read-only):
  - фильтр по клиенту, дате, теме
  - быстрый переход в конкретную сессию / review bad answers
- Метрики качества:
  - % ответов с 👍/👎 по клиентам и по темам
  - самые частые "плохие ответы" в разрезе клиентов
- Управление рисками:
  - видеть у кого нет ENCRYPTION_KEY / OpenAI key
  - видеть аномальный трафик по token usage

**Безопасность:**
- Отдельная роль admin (не обычный client user)
- Отдельный login / флаг в JWT

**Приоритет:** P2 для тебя лично (чтобы управлять платформой), но может подождать пока не появится 2+ клиентов.
