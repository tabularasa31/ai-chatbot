# Chat9 Demo Docs

Эти документы предназначены для загрузки в демо-аккаунт Chat9,
чтобы виджет на лендинге мог отвечать на вопросы о самом продукте.

## Как использовать

1. Создать отдельный аккаунт на getchat9.live (например, demo@getchat9.live).
2. Добавить OpenAI API key.
3. Загрузить все `.md` файлы из этой папки.
4. Создать эмбеддинги для каждого документа.
5. Скопировать API key этого аккаунта.
6. Вставить в `cursor_prompts/landing_page.txt` вместо `DEMO_API_KEY`.
