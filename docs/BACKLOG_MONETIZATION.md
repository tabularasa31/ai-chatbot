# Monetization Backlog

Идеи по монетизации Chat9.

---

## Модель: Freemium

Базовая модель — бесплатно с ограничениями, платно без.

### Free tier (текущее состояние)
- До 20 документов.
- Все базовые фичи (виджет, логи, feedback).
- "Powered by Chat9" в подвале виджета.

### Premium (идеи для платного тарифа)

| Фича | Free | Premium |
|------|------|---------|
| Документы | 20 | Безлимит / 100+ |
| Убрать "Powered by Chat9" | ❌ | ✅ |
| Chat sessions per month | 500 | Безлимит |
| Кастомный system prompt | ❌ | ✅ |
| Приоритетная поддержка | ❌ | ✅ |
| Branded виджет (кастомные цвета) | ❌ | ✅ |
| Multiple team members | ❌ | ✅ |
| API access (программный upload) | ❌ | ✅ |
| Zendesk/Intercom интеграция | ❌ | ✅ |

---

## Идеи по тарифам

### Вариант A: По сессиям (usage-based)
- Free: 500 чат-сессий/месяц.
- Starter ($19/mo): 2000 сессий.
- Pro ($49/mo): безлимит.

**Плюс:** легко понять, легко масштабировать.
**Минус:** клиент не знает сколько сессий у него будет.

---

### Вариант B: По клиентам / доменам
- Free: 1 домен / 1 клиент.
- Pro ($29/mo): до 5 доменов.
- Business ($99/mo): безлимит доменов.

**Плюс:** хорошо для агентств/реселлеров.

---

### Вариант C: Flat monthly (простой)
- Free: ограниченный набор.
- Pro ($29/mo): всё включено.

**Плюс:** самый простой для клиента.
**Минус:** нет промежуточных точек апсейла.

---

## Вирусные механики

### [FI-038] "Powered by Chat9" в виджете
- Free → брендинг виден.
- Premium → убрать (стимул апгрейда).
- Каждый виджет = реклама Chat9 на сайте клиента.

### Referral program (идея)
- Клиент приглашает другую компанию → получает бесплатный месяц Premium.
- Реферал получает скидку 20% на первые 3 месяца.

### Public "Built with Chat9" badge
- Клиент может добавить бейдж на свой сайт.
- Ссылка на Chat9 в открытом виде.

## Что делать сейчас

1. Оставить Free без лимитов — пока нет достаточной базы пользователей.
2. Внедрить FI-038 ("Powered by Chat9") — это вирусный маркетинг.
3. Когда появятся 5–10 активных клиентов — провести интервью: за что готовы платить?
4. После интервью — выбрать модель тарификации и запустить Stripe.
