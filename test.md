# AI Chatbot Platform FAQ

## What is this platform?
This is an AI-powered chatbot platform that answers questions based on your documentation.

## How to reset password?
Go to Settings -> Security -> Click "Reset Password" -> Check your email.

## What file formats are supported?
We support PDF, Markdown (.md), and Swagger/OpenAPI (.json, .yaml) files.

## How much does it cost?
The MVP version is free. Premium plans start at $19/month.

## How to embed the widget?
Add this to your website HTML:
<script src="https://api.yourdomain.com/embed.js"></script>
<div id="ai-chat-widget"></div>
